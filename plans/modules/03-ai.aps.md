| ID | Owner | Status |
|----|-------|--------|

## Purpose

Add optional AI-powered enrichment to link cards. Provide a provider abstraction supporting multiple AI services and a "Summarize Link Card" command that generates TLDR, summary, and topic tags from page content.

## In Scope

- AI provider abstraction (common interface)
- OpenAI, Anthropic, and GitHub Copilot provider implementations
- Text extraction from fetched HTML content
- "Summarize Link Card" command
- Prompt construction for TLDR, summary, and tag generation

## Out of Scope

- AI provider settings UI (handled by CORE -- settings interface)
- Card rendering of AI fields (handled by CORE -- card renderer already supports tldr/summary/tags)
- Local/offline AI models

## Interfaces

**Depends on:**

- CORE -- settings interface for provider config, metadata fetcher for page content

**Exposes:**

- AI provider interface (extensible for future providers)
- "Summarize Link Card" command registered on the plugin
